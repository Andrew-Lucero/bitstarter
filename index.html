Hello! I am developing a standard game of chess. Who doesn't love chess? There are several different options you can select in the main menu, including local player vs. player, pvp across the internet, the ability to play against an AI that is being optimized at the moment, the ability to watch replays of past games, and even an option to watch two AIs duke it out! More features may be added (or possibly taken away!) if there is enough support (or complaints).

All the action is beautifully rendered in a command line, showing great detail; if you are at risk for seizures, you may want to consult a physician before playing. Currently the game is only available on Windows platforms, but if there is enough support a Linux port will be made as well. Further, if too many people can't handle the intensity of playing on a command line, a GUI will also be made, along with making the game overall more user friendly. A barebones version of the game will be made even if the goal is not reached, but if the goal is reached more features will be added and the game will end up extremely polished! Thanks for the support!
